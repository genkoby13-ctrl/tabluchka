# Таблиця x÷ Тренажер

Додаток для тренування таблиці множення та ділення на Kivy.

## Вимоги

- Python 3.8 - 3.12 (Kivy не підтримує Python 3.14)
- Kivy >= 2.0.0

## Встановлення

### Якщо у вас Python 3.14:

Kivy ще не має підтримки для Python 3.14. Вам потрібно:

1. Встановити Python 3.11 або 3.12 з [python.org](https://www.python.org/downloads/)
2. Або використовувати віртуальне середовище з Python 3.11/3.12

### Встановлення залежностей:

```bash
pip install -r requirements.txt
```

Або безпосередньо:

```bash
pip install kivy
```

## Запуск

```bash
python main.py
```

## Тестування

Перевірте, чи все працює правильно:

```bash
python test_app.py
```

Цей скрипт перевіряє:
- ✓ Синтаксис коду
- ✓ Структуру класів
- ✓ Наявність необхідних модулів

## Використання

1. Виберіть діапазон чисел (від і до)
2. Оберіть операції (множення та/або ділення)
3. Натисніть "РОЗПОЧАТИ ТРЕНУВАННЯ!"
4. Вводьте відповіді на приклади
5. Перегляньте результати після завершення

## Створення APK для Android

Для створення APK файлу дивіться інструкції в файлі **BUILD_ANDROID.md**

**Коротко:**
- Потрібен Linux (WSL, Docker або VM)
- Встановіть Buildozer
- Виконайте: `buildozer android debug`

Детальні інструкції: [BUILD_ANDROID.md](BUILD_ANDROID.md)
